# 🔍 聊天记录搜索
<br>
为SillyTavern设计的轻量插件。
版本：1.5.0 |  作者：Thirteen-Moons

---

## ⭐ 功能
- 通过关键词搜索历史楼层
- 快速回顶回底
- 快速跳转指定楼层
- 查看指定消息相邻上下文
  
---

## ⚙ 面板说明
**快捷按钮位于 输入框左下方魔棒菜单中，*「聊天记录搜索」***
<br><br>

### 📍 横条快捷面板
  点击「聊天记录搜索」，弹出快捷键面板。
  
- **第一个按钮**：搜索聊天记录
- **第二个按钮**：快速回顶
  >此顶部取决于你设置加载了多少楼层消息；若需回到0层，请在搜索页中输入0进行跳转
- **第三个按钮**：回到底部
<br>

### 💾 搜索面板
- **跳转**：输入楼层可跳转至对应楼层
- **搜索**：输入关键词跳转至 *「搜索结果页」*
  > 支持多个关键词联合搜索，每个关键词以一个空格分隔
<br>

### ✅ 搜索结果页
- **关键词高亮**：以补色高亮，任意主题均美观
- **搜索出的消息跳转**：点击搜索出的楼层对话，可直接跳转至相应楼层
- **复制**：复制该楼层消息
- **预览相邻楼层**：点击跳转至 *「预览页」*，可浏览该楼层的相邻楼层上下文（丝滑）
<br>

### 📃 预览页
- 可滑动查看消息，点击右上角按钮可复制该楼层消息。

## 🖊 更新日志
### v1.2.0
- **功能改进**：在楼层跳转时临时解除SillyTavern的加载限制，楼层跳转不再受此设置 或 *隐藏助手* 的设置影响。
  > 使用回底按钮，即可恢复加载限制。

- **外观优化**：将浏览器原生提示改为弹窗提示
<br>

### v1.4.0
- **整体架构重构**：将搜索页、结果页、预览页拆分
- **新增预览页**：无需跳转即可丝滑查看相邻楼层
- **移动端显示优化**：移动端现可随软键盘弹出自动上缩，不再遮挡输入框。
- **跳转逻辑优化**：由原来的临时加载全部消息，改为计算需要加载的最小值，减少跳转卡顿。

### v1.5.0
- 整体性能优化
