# 🔍 聊天记录搜索
<br>
为SillyTavern设计的轻量插件。
版本：1.2.0 |  作者：Thirteen-Moons

---

## ⭐ 功能
- 通过关键词搜索历史楼层
- 快速回顶回底
- 快速跳转指定楼层
  
---

## ⚙ 面板说明
**快捷按钮位于 输入框左下方魔棒菜单中，*“聊天记录搜索”***
<br><br>

### 📍 横条快捷键
  点击“聊天记录搜索”，弹出快捷键面板。
- 第一个按钮：搜索聊天记录
- 第二个按钮：快速回顶
  >此顶部取决于你设置加载了多少楼层消息；若需回到0层，请在搜索面板中输入0进行跳转
- 第三个按钮：回到最新消息的开头
- 第四个按钮：回到最新消息的结尾
<br>

### 💾 搜索面板
- **跳转**：输入楼层可跳转至对应楼层
- **搜索**：输入关键词可显示含有包含关键词的楼层
- **复制楼层**：点击可复制楼层内容
- **搜索出的消息跳转**：点击搜索出的楼层对话，可直接跳转至相应楼层
  <br><br>
